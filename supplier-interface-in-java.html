4. Supplier

The Supplier functional interface is also a type of functional interface that does not take any input or argument and returns a single output. On calling the supplier it simply returns a value. The supplier takes only one generic type, the type of data it is going to return. get() is the abstract method of the Supplier. 

public static void main(String[] args) { 
    Supplier<String> supplier = () -> "Hello World";
    System.out.println(supplier.get());        
};

Output:
Hello World
